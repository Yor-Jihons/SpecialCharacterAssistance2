# SpecialCharacterAssistance2

## はじめに

このアプリは特殊文字を入力しやすくするためのものです。

## コンセプト

スペイン語の`ñ`(チルダ付きn)といった特殊文字の入力が大変です。
なので、このアプリで通常の入力をしつつ、ボタンでも入力できるようにしました。

## プロジェクトの構成

テストは`projnew/main.xUnits`ディレクトリ内にあり、本番プロジェクトは`projnew/main`ディレクトリ内にあります。

## 開発プロセスにおけるAIの活用

当初、Windows固有のイベント処理（ショートカットキー等）の実装においてコードビハインドが肥大化する課題がありました。MVVMパターンの純粋性を保ち、テストコードを記述可能にするため、AI(Gemini CLI)をペアプログラマとして活用。イベントをICommandへ変換するリファクタリングを効率的に行い、ロジックをViewModelに集約させることで、単体テストが可能な設計を実現しました。

## エラー思想

予期せぬエラーが発生した場合、不正な状態で処理を継続しユーザーのデータを破損させるリスクを避けるため、意図的にFail-Fast（速やかな異常終了）を選択しています。ただし、Logger.csを実装しており、クラッシュ時の情報をログに残すことで、開発者が後から原因究明できるような最低限のトレーサビティは確保しています。
